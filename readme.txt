Résumé : pour partir l'application il faut ajouter le projet tp3 et exécuter le main de SessionFrm qui se trouve à cet endroit : edu.uqam.inf5153.tp3.presentation.SessionFrm.

Pour y arriver, 
  Ouvrir le projet tp3 :
	- Dans Eclipse, File/Open Project from File System..., cliquer sur Directory, aller dans le répertoire inf5153-tp3 et sélectionner tp3 faire ok et faire suivant.
	- faire afficher le Project Explorer, cliquer sur src/main/java, 
		cliquer sur edu.uqam.inf5153.tp3.presentation,
		cliquer sur SessionFrm.java
		cliquer sur SessionFrm
		cliquer bouton droit sur main et sélectionner Run As > Java Application

Le projet devrait démarrer à ce moment, une fenêtre demande d'entrer ou de créer un utilisateur. 

Utilisateur:
Entrer l'utilisateur de votre choix en appuyant sur créer nouvel utilisateur puis vous pouvez le créer.
Pour entrer un médecin existant, entrer utilisateur : caro et mot de passe : caro et appuyer sur "Entrer".
Pour un utilisteur personnel médical, entrer utilisateur : persomed et mot de passse : persomed et appuyer sur "Entrer".
Il est aussi possible de créer l'utilisateur de votre choix en cochant ou décochant Personnel médical et en appuyant sur "Créer un nouvel utilisateur". 

Dossier médical:
Lorsqu'on demande d'entrer le dossier médical, il faut entrer "abc1234" qui est un dossier existant et appuyer sur "OK".

Le dossier médical s'affiche ici.  Si on est un médecin, il est possible d'ajouter une visite en appuyant sur le bouton "Ajouter une visite", 
un antécédent en appuyant sur le bouton "Ajouter un antécédent" ou de modifier tous les champs de la page exepté le numéro d'assurance maladie.
Si c'est un personnel médical, il peut seulement consulter tout le dossier, mais sans pouvoir modifier les champs.
